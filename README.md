# React-Non-CRA
Clean, performant, feature-complete TypeScript React project template, built without CRA. No React Scripts, no unused code and no black boxes, just pure React ❤️

## Features
- React 18
  - Full TypeScript support.
  - Optimized `.tsconfig.json`.
  - HMR using **[React Refresh]()**.
- Fully customizable Babel and Webpack configs.
  - Full JSX support using **[@babel/preset-react](https://babeljs.io/docs/en/babel-preset-react)**.
  - ES6+ support using **[@babel/preset-env](https://babeljs.io/docs/en/babel-preset-env)**.
  - Performant HTML and CSS bundling and optimization using **[HtmlWebpackPlugin](https://webpack.js.org/plugins/html-webpack-plugin/)** and **[style-loader](https://webpack.js.org/loaders/style-loader/)** + **[css-loader](https://webpack.js.org/loaders/css-loader/)**.
- Modern ESLint and Prettier configs.
  - Industry standard optimized style guides for keeping your codebase maintainable.
  - Prettier format-on-save support (VSCode).
